🚀 Login UI — Minha Loja

Interface moderna de login feita em HTML, CSS e JavaScript puro, com foco em experiência visual premium, animações suaves e feedback interativo.

✨ Demonstração

Tela de login com:

Fundo animado com orbs e partículas
Card com efeito 3D no hover
Input estilizado com ícones
Botão com estado de loading
Toast de erro
Tela de sucesso animada
Social login (Google e Facebook UI mock)
🎯 Objetivo

Criar uma experiência de login que não pareça “mais uma tela de login genérica da internet”, mas sim algo com identidade visual moderna, fluida e agradável.

Ou seja: login que não dá vontade de fugir dele.

🧱 Tecnologias usadas
HTML5
CSS3 (com animações avançadas e glassmorphism)
JavaScript Vanilla
SVG icons inline

🎨 Destaques visuais

🌌 Fundo animado
Orbs coloridos com blur
Partículas flutuantes dinâmicas

🧊 Card principal
Efeito glassmorphism
Bordas suaves e sombras realistas
Animação de entrada suave

🧠 Interações inteligentes
Mostrar/ocultar senha
Validação básica de inputs
Toast de erro
Loading no botão de login
Tela de sucesso simulada

🎮 Microinterações
Tilt 3D no card
Hover com glow
Transições suaves em tudo (sim, tudo mesmo)
